# 🚀 Spring Boot Tutorial

A structured collection of Spring Boot projects covering core concepts to advanced topics. This repository documents my learning journey through Spring Boot by implementing each concept in a separate project.
## 📌 About

This repository contains hands-on examples and mini-projects for learning Spring Boot. Each folder represents an independent Spring Boot project focused on a specific concept.

The goal is to build a strong understanding of Spring Boot fundamentals, backend development, and industry best practices.
